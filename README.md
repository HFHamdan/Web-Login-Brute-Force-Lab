# Web-Login-Brute-Force-Lab
Built a Flask login system and brute-force attack script to demonstrate credential-based attack and defense techniques.

# Overview
Built a local web login application using Python and Flask, then developed and executed a brute-force attack script against it to demonstrate credential-based attack techniques. This project was completed entirely locally with no cloud infrastructure, focusing on the practical mechanics of how brute-force and credential stuffing attacks work and how to defend against them.

# What Was Built

- A functional local login page mimicking a real-world web application authentication flow
- A username and password wordlist for testing
- A brute-force attack script that systematically cycles through credential combinations until successful authentication is achieved
